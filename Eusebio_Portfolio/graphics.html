Nike Site Evaluation

Site evaluated:

Purpose of graphics on this site:

The Nike site look very clean and has a lot of easy movements for first time users. It has many video animations which make the website a lot more exciting.

Guidelines for effective use of graphics

  1. Yes - Graphhics fit the purpose, orginization, and style of the page.
  2. Yes - Large graphics have been avoided.
  3. Yes - Graphics help to guide the viewers focus to the important content on the page.
  4. Yes - Overly bright and potentially obnoxious images have been avoided.
  5. Yes - Graphics have not been used for textual content, other than the banner.
  6. Yes - Alternate text is present for all images.
  7. Yes - Where text appears in graphics (the banner) there is ample contrast.

Graphic File Types

  1. png - because of the colors being flat.
  2. jpeg - because it has a lot of colors.
  3. Jpeg - because it has several different colors.
  4. png - because it has a large flat area.
  5. png - because it its a solid color.
